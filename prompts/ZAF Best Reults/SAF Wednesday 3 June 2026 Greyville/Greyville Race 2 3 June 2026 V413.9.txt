Mathematical Formula Updates

To resolve these errors, we introduce the Surface Transition & Layoff Decay
Coefficient (\Omega):

\Omega = \left( 1 - \beta \cdot e^{-\frac{D_{\text{lay}}}{30}} \right) \cdot \left( 1 - \gamma_{\text{surf}} \right)

Where:

  - D_{\text{lay}} = Days since last run (e.g., 70 days for #02).
  - \beta = Layoff decay constant (set to 0.15).
  - \gamma_{\text{surf}} = Surface experience penalty factor (0.20 if first run
    on synthetic; 0.00 if experienced).

This modifier scales down the horse's projected Terminal Acceleration
(\text{TAT}_{\text{proj}}):

\text{TAT}_{\text{adj}} = \text{TAT}_{\text{proj}} \cdot \Omega

Programmatic Implementation Draft

def calculate_omega(days_layoff, surface_experience):
    beta = 0.15
    gamma_surf = 0.20 if not surface_experience else 0.00
    
    # Calculate layoff decay component
    layoff_factor = 1.0 - (beta * (2.71828 ** (-days_layoff / 30.0)))
    
    # Calculate surface transition penalty
    surface_factor = 1.0 - gamma_surf
    
    return layoff_factor * surface_factor
