PHASE 8: SOVEREIGN KINETIC INDEX (SKI) ENGINE

Section 1: Baseline Ingestion & Metrical Anchors

To prevent static form traps, all mathematical evaluations of raw capacity are
consolidated into the Sovereign Kinetic Index. This continuous efficiency index
acts as a baseline "Raw Engine Capacity" index (SKI_i), which is subsequently
scaled to compute the Net Thermodynamic Conservation Index (NTCI_i).

Section 2: Raw Nominal Latent Speed (Raw NLS)

Calculate base speed potential:

1.  Base Scalar: Initialize with a standard baseline of 80.
2.  Career Class Adjustments: Add up to +10 based on career win/place
    percentages and class competitiveness. Deduct up to -10 for high-mileage,
    non-competitive profiles (Win Rate <5\%).
3.  Recent Performance Peak: Add up to +5 for strong closing sectionals or
    close-margin finishes (<1.5L) in higher grades.
4.  Ex-Import Class Drop Override: If an international/provincial import drops
    class: Add a Class Parity Exemption Score
    (\text{CPX} = \text{Class Drop Delta}). If \text{CPX} \ge 2.50, add +5
    directly to Raw NLS. Apply Regional Gravity Shift Modifier (LCDM): If
    \text{Circuit\_prior} \in \{\text{Kenilworth, Durbanville}\} AND
    \text{Circuit\_current} \in \{\text{Turffontein, Vaal}\} \implies LPP_i = LPP_i + 2.0.
5.  Synthetic-Turf Decoupling: If today's track is Synthetic (Silo D), decouple
    turf form entirely:
    \text{Raw NLS}_{\text{synthetic}} = (0.70 \times \text{NLS}_{\text{synthetic\_historic}}) + (0.30 \times \text{NLS}_{\text{recent\_overall}}).
6.  Sophomore Stamina Stability (SSS) Class Override: If Age == 3\text{YO} AND
    target distance \ge 2000 m AND track condition is Soft/Heavy
    (MSCI_{\text{eff\_i}} \le 0.72) AND the runner possesses a prior win at
    \ge 2000 m on Soft/Heavy ground, strictly reduce any applied negative Class
    Gravity penalties by 40%.
7.  Castration Hormonal Recoil Adjustment: Multiply Raw NLS by PGRF_i
    (Formula 53).
8.  Detraining Aerobic Decay Adjustment: Deduct CRDI_i (Formula 54) directly
    from Raw NLS:
    \text{Raw NLS}_{\text{calibrated}, i} = \text{Raw NLS}_i \times PGRF_i - CRDI_i.
9.  Unexposed Class-Ceiling Multiplier (UCCM) Sparing: If unexposed 2YO with
    elite pedigree, apply +3.0 NLS bonus:
    \text{Raw NLS}_{\text{calibrated}, i} = \text{Raw NLS}_{\text{calibrated}, i} + \gamma_{\text{pedigree}} \times (3.0 - Starts).

Section 3: Twelve Modular Latent Frailty (\nu_i) Adjustments

Baseline frailty \nu_{\text{base}} = 0.50. Calculate the final \nu_i by applying
these adjustments:

1.  Cardiorespiratory Recovery Decay (\Delta\nu_{\text{decay}})
      - Rule 1.1 (Continuous Recovery Decay): If second-up off a layoff of
        t_{\text{layoff}} \ge 90 days, calculate second-up decay using
        Formula 18 (SURD_i). On turf rated Soft 6 or worse
        (MSCI_{\text{eff}, i} \le 0.75), this decay penalty is mandatory and is
        multiplied by 1.50.
      - Rule 1.11 (Distance-Scaled Spell Decay - CDDS): For short-course races
        (D \le 1000 m), layoff decay is scaled down by f(D):
